# Tetris Touch

Canvas 기반 모바일 웹 테트리스입니다. 화면 안에 조이스틱과 게임 버튼을 모두 배치해 휴대폰 브라우저에서 바로 플레이할 수 있습니다.

## Features

- TypeScript + Vite 기반 정적 프론트엔드
- Canvas 2D 렌더링 테트리스 보드
- 모바일 화면 최적화 레이아웃
- 화면 내 조이스틱: 좌/우/소프트드롭/회전
- 게임 버튼: DROP, PAUSE, NEW
- 키보드 지원: 방향키, Space, P, R
- Vitest 기반 핵심 게임 로직 테스트

## Development

```bash
npm install
npm run dev
npm test
npm run build
```
